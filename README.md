# 조작
W,A,S,D (이동)  
  
마우스 좌클릭 (공격)  
  
마우스 우클릭 (조준 - GunCharacter)  
  
Q,R (스킬사용)  
  
E (회피)  

Shift (달리기)  
  
SPACE (점프)  
  
F1, F2, F3 (캐릭터 전환)
